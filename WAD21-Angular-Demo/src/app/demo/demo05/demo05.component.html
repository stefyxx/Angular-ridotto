Pipe sur les strings

Valeur de "maVariable1" : {{ maVariable1 }}

avec Pipe "maVariable1" : {{ maVariable1 | lowercase }}

Pipe sur les nombres

Valeur de "maVariable2" : {{ maVariable2 }}

avec Pipe "maVariable2" : {{ maVariable2 | number : '3.3-5' }}

Pipe sur les dates

avec Pipe "maVariable3" : {{ maVariable3 | date }}
